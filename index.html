How to use Screenreader on MacOS

Ronal Gomez and Thomas Brickhouse - CS 343 - Dr. Stewart
Find Documentaion on Screenreader here

Disclaimer: Our tests of screenreader and voice helper were used on the operating system MacOS. We both tested on the Google Chrome Browser using MacOS 13.0.1.

Sites with Screenreader

  • Youtube

    • Youtube was very simple to navigate from video to video using the tab and shift tab shortcuts. Clicking into the videos was simple as the shortcut that was only neccessary was hitting the enter button.

  • ESPN

    • ESPN was a horrible experience because it didn't follow a good path to be able to tab from link to link. If you wanted to skip over from one link to another it would take an extra short cut to find what you were looking for. Infortmation was too cluttered and heavy to find.

Screenreader Keyboard Shortcuts

NOTE: VO is the Equivilant to Control + Option

Task Command
Command + Fn + F5 Turn Screen Reader On
Command + Fn + Double tap F5 Turns off Screen Reader
Control Toggle reading on and off
VO + U Opening links on your webpage that can be navigated with the up and down arrows
VO + Space Activate Link or Button
VO + Right or Left Read next or previous item
VO + Z Repeat last item read
Command + L Browser Search Bar
VO + Command + H Reads next heading
VO + Command + L Next Link
VO + Command + I Next Image
VO + Command + J Next Form Control

First time using Talkback

It's hard to understand how to do anything at first. Even with being able to look at the screen it still brought difficulty being able to navigate with Screenreader turned on. When voice is turned on, it will automatically cycle through the links and read them out to you.

Using Screenreader without vision

The biggest difficulty that came with not being able to see was being able to find exactly where all of the keys were on the keyboard. Not knowing where everything was made it difficult to be able to do the short cut keys. While it's talking to you, it makes it difficult to focus on anything other than the loud voice being played through to your ear. Surrounding information become obsolute.
